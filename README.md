# Solo Rosas Website

A beautiful, elegant website for Solo Rosas - Premium roses cultivated at high altitudes in Ecuador.

## Project Structure

```
SoloRosas/
├── index.html              # Home page with featured white rose
├── collection.html         # Collection page with all rose varieties
├── style.css              # Styling for both pages
├── roses-data.js          # Rose varieties data
├── assets/
│   ├── WhiteRose.png      # Main hero rose image
│   ├── WhiteRoseNavBar.png # Navigation bar rose stem
│   └── roses/             # Individual rose images (to be added)
└── README.md
```

## Features

### Home Page (`index.html`)
- Full-screen layout with elegant white rose
- "SOLO ROSAS" brand name on the left
- Navigation menu on the right
- Responsive design
- Click "OUR COLLECTION" to navigate to collection page

### Collection Page (`collection.html`)
- Rose stem navigation bar at the top
- Grid display of 9 premium rose varieties
- Hover effects on rose cards
- Responsive layout (3 columns on desktop, 2 on tablet, 1 on mobile)

## Adding Rose Images

To complete the collection page, add individual rose images to `assets/roses/`:

1. `all-for-love.png`
2. `aquarello.png`
3. `asos.png`
4. `aloha.png`
5. `absolut-pink.png`
6. `altamira.png`
7. `amadea.png`
8. `antonia-garden.png`
9. `art-deco.png`

Place these images in the `assets/roses/` folder and the collection page will automatically display them.

## How to Use

1. Open `index.html` in a web browser to view the home page
2. Click "OUR COLLECTION" to navigate to the collection page
3. Click the Solo Rosas logo to return to the home page

## Design Features

- Elegant serif fonts (Georgia) for branding
- Warm color palette (beige, tan, soft whites)
- Smooth hover transitions
- High-quality responsive design
- Premium aesthetic matching luxury rose brand

## Customization

- Edit `roses-data.js` to add/modify rose varieties
- Update colors in `style.css` (current palette uses `#a67c52` for accents)
- Modify navigation links in HTML files
